## Installation & Setup
```sh
 git clone https://github.com/Alph702/Blog.git
 cd Blog
```

```sh
 pip install uv
```

**For dev**:
```sh
 uv sync
```
**For use**:
```sh
 uv sync --no-dev
```

```sh
 uv run app.py
```
